# FSD Project Manager

A minimal web project for Full Stack Development lab (EXP06).

## Contents
- `index.html`: Main HTML page for the experiment.

## Getting Started
Open `index.html` directly in your browser, or use a simple static server.

### Option 1: VS Code Five Server
- Start the Five Server extension and open `index.html`.

### Option 2: Python HTTP server
```powershell
# From the project folder
python -m http.server 5500
# Then open http://localhost:5500/index.html
```
